# neon-drift
Neon Drift - 霓虹漂移  A neon-style space shooter game. Control your ship, destroy enemies, collect stars, and survive the waves!  Features: - Neon visual style - Multiple enemy types - Wave-based challenges - Upgrade system - Background music &amp; sound effects  Controls: - Mouse/Touch to move - Click/Touch to shoot - ESC to pause - Q to use bomb
